# projeto-individual-react-procurados-one-piece
Projeto individual desenvolvido em **React** simulando o sistema de cartazes de recompensa e central de denúncias da Marinha do universo de One Piece. 

## Sobre o Projeto
Esta aplicação foi construída para colocar em prática conceitos fundamentais de desenvolvimento frontend com React. O projeto conta com duas telas principais (Lista de Procurados e Central de Comunicações) que conversam entre si de forma dinâmica, trazendo uma experiência imersiva baseada no anime.

## Tecnologias Utilizadas
* **React**
* **React Router Dom**
* **Styled-Components** 
* **JavaScript**

## Como executar o projeto



1. Clone este repositório no seu computador:
```bash
git clone https://github.com/LeiltonBraga/projeto-individual-react-procurados-one-piece.git
```

2. Entre na pasta do projeto:
```bash
cd projeto-individual-react-procurados-one-piece
```

3. Instale as dependências do projeto, incluindo React Router e Styled-Components:
```bash
npm install 
npm install styled-components
npm install react-router-dom
```

4. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acesse o link gerado no terminal (geralmente \`http://localhost:5173`) no seu navegador ou dê um Crtl + click para abrir diretamente.
